# jade-all
Jade,是Java应用程序访问Mysql数据库的一个小框架.

**数据库访问流程图**
+-------------+           +------------+          +------------+           +-------------+ 
+JAVA应 用 程 序       +           +Jade DAO接 口    +          +   Jade 框 架     +          +数 据 库 服 务 器            +
+-----+-------+           +-----+------+          +-----+------+           +------+------+
      |                         |                       |                         |
      |1.调 用 已 实 现 Jade 规 范 的 DAO                                                         
      +-----------------------> |                       |                         |
      |                         |2.处 理 DAO 服 务                                |                         |
      |                         +---------------------> |                         |
      |                         |                       |3.提 交 请 求 到 数 据 库                              |
      |                         |                       +-----------------------> |
      |                         |                       |                         |
      |                         |                       |                         |
      |                         |                       |  4.返 回 查 询 结 果                                  |                                
      |                         |5.按 DAO 规 范，将 数 据 库 对 象    <-----------------------+ |
      |                         | <---------------------+                         |
      | 6.返 回 查 询 结 果                                     |     对 象 包 装 成 JAVA 对 象     |                         |
      | <-----------------------|                       |                         |
